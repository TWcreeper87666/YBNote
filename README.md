# YBNote

在立體的空間中擺放你的音符，拼湊出你的音樂！

基於麥塊的極限是 0.05 秒，因此實際的體驗並不是很舒服。

Inspired by: BOCLAA

### TOOLS

- `yb:eui_open` 右鍵開啟介面，可修改、創建音符
- `yb:note_play` 右鍵播放音符，左鍵調整音高
- `yb:note_edit` 右鍵移動、創建音符，左鍵移除音符
- `yb:note_record` 開啟錄製選單
